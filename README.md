# ITVAL — Sitio web corporativo

Sitio bilingüe (ES/EN) para ITVAL, especialistas en aluminio y vidrio.

## Requisitos

- Node.js 18.18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) — redirige a `/es`.

## Producción

```bash
npm run build
npm start
```
